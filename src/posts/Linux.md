# Linux

## 命令基础

+ 命令行：Linux终端(Terminal)，是一种命令提示符页面。以纯字符的形式操作系统，可以使用各种字符命令对系统发出操作指令
+ 命令：Linux程序，一个命令就是一个Linux的程序。命令没有图形化的页面，可以在命令行提供字符化的反馈
